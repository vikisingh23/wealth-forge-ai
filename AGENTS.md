# FinForge AI Agents

This directory contains the specialized intelligence profiles for the personal finance team.

## Core Utilities
- **FinForge Unified Hub**: Central intelligence server managing custom financial "Blades" (Portfolio, Policy, Tax).
- **FinWorth Math Engine**: High-precision JS/TS library for raw financial math.
- **Office-MCP (@neuraforge/office-mcp)**: Professional document processing.

## The Workforce (Team FinForge)

### 🧠 The Brain
1.  **Lead Financial Planner**: Synthesizes all data into a master roadmap. Orchestrates other agents.

### 👤 The Profile
2.  **User Profile Manager**: Collects and manages age, city, salary, and personal goals safely.

### ⚖️ The Specialists
3.  **Tax Strategist**: Expert in Income Tax Act, GST, and tax-efficient planning.
4.  **Mutual Fund Specialist**: Deep-dive analysis into equity/debt mutual fund portfolios.
5.  **Policy Analyst (LIC/ULIP/GWP)**: Audits insurance contracts and benefit plans for true value/IRR.

7.  **Budget & Cashflow Agent**: Tracks spends, builds emergency funds, and generates Excel sheets.
8.  **Policy Scout**: Monitors RBI, SEBI, and global news for impact on personal finance.
